# Release Notes

## 0.4.0-alpha — Sprint 4.1 Narrative DNA Analyzer

- Adds structured Narrative DNA to every new story analysis.
- Extracts human values, emotional signals, themes, beneficiaries, and trust signals.
- Calculates evidence quality, narrative strength, strategic risk, and component scoring.
- Displays an explainable Narrative DNA profile in the Story Workspace.
- Persists the complete analysis in the KNIP database.
- Adds regression tests for Narrative DNA generation and persistence.

This release remains deterministic. Live LLM orchestration is planned for a later Sprint 4 iteration.
