# KNIP v0.3.0 Alpha — Story Intelligence Workspace

## Added
- Rich story intake metadata and full narrative capture
- Five-part strategic scoring: human impact, strategic value, credibility, opportunity, and risk
- Explainability panel with improvement guidance
- Deterministic audience matching with audience-specific rationale
- Story review status workflow and approval action
- Expanded evidence provenance fields
- Audience API and expanded automated tests

## Notes
This release remains local-first and deterministic. No external LLM or cloud service is required.
